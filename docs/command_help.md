**群聊环境下，同一个群同一个人同一条指令有10秒cd，以期望解决多个机器人相互嵌套的问题**

***因为指令是可以自定义的，因此这里以我自己定义的指令来说明***

**参数介绍: 小括号内的`()`为可选参数，方括号`[]`内的是必选参数，花括号`{}`内的是可用的参数,尖括号`<>`内的是参数类型**

***<font color='gree'>请注意，并不是所有的指令都会列在这里，写的插件越来越多，我也会忘记更新文档，很多指令的用法我一个文档也写不清楚</font>***

# 指令面板

```
 help / 帮助    
```
使用 `help`或者`帮助`来呼出指令面板

# 私聊命令(管理员)

## PING
```
ping
```
向服务器ping一下，来验证机器人是否在线


## 管理员控制

```
addadmin / deladmin qq号
```
分别是 添加机器人管理员 和 删除机器人管理员  **<font color="blue">仅有机器人主人有该权限</font>**

## 获取机器人配置
```
getinfo
```

获取机器人当前设置

## 获取日志(未完成)

```
log (level) (num)
```
获取最近 `num` 条的等级为 `level` 的日志 (未完成，以后再去完善)

## 让机器人发送文本

```
stext:: [群号] [文本]
```

向指定群发送文本

## 黑名单控制

```
addblacklist qq号
```
将一个用户添加进黑名单

```
delblacklist qq号
```
将一个用户从黑名单移出

# 群命令(管理员)

## help

同上

```
silence on/true/任意
```
开启全局沉默模式，尽可能减少信息输出,其他参数时为关闭沉默模式

## 群复读管理

```
norepeat on/true/任意
```
参数为 `on` 或 `true` 时，禁用机器人的复读功能，其他参数为启用复读

## 减少信息输出

```
silence on/true
```
开启该群的沉默模式，尽可能减少信息输出

其他参数时为关闭改群沉默模式


## roll点 / 随机数 / 抽词

```yaml
roll                  # 返回 0-100 随机数
roll 90               # 返回  0-90 随机数
roll 114514-1919810   # 返回 114514-1919810 之间的随机数
roll 可爱 的 拉克丝    # 返回 随机一个词 ("可爱","的","拉克丝")
roll d6               # 掷一个 n 面的骰子
```

掷筛,随机数,随机抽词

## 向机器人主人发送消息

```
转告主人 [文本]
```

在群聊中,别人可以通过发送这段文本，来给bot的主人发消息

**<font color='FF0000'>黑名单用户无法使用此功能 ！</font>**

## 签到和积分

```
签到
```

获取1点积分  

*安装 Tarot 插件时会抽一张塔罗牌*

```
获取当前积分
```

获取当前积分

## 天凤相关

### 查分

```
thpt / 天凤分数 / 天凤pt   [玩家名] {reset=True}
```
查天凤分数 

该功能存在小问题,无法判断账号分数是否会重置,默认是超过180天就会重置。如果不重置请追加 `reset=false` 

### 添加关注
```
thadd [玩家名]
```
添加一个玩家进入天凤对局查询队列，有新对局时会播报

### 移除关注

```
thdel [玩家名]
```

将一个玩家从天凤对局查询队列中删除

### 获取全部关注

```
thgetwatch
```
获取本群天凤关注

### 牌理分析(存在问题)

```
牌理 [牌型文本] # 例如： 114514p1919s810m1z
```

天凤牌理分析 ( 不含国士无双、七对子)

## 图片生成器

###  举牌
```
举牌 [内容] 
```

将内容写在举牌小人上发出来，最多支持40个字符

###  亲、吃、举、逮捕、击杀  *etc.*
```
亲/亲亲 @群友
```

生成一个互亲的图

*丢、吃、举、逮捕、击杀 等等指令同理*

### 摸头

```
摸/摸摸 @群友
【戳一戳】
```

发送一张摸头图

### 黑白图生成

```
bw [文字] [图片]
```
返回一张黑白化的图，并且底部配有文字

### 截图

```
截图 [<回复消息>]
```

将聊天记录文本进行截图 **只能截图文本**

## 色图

仅有机器人管理员有权限控制色图开启

### 功能开关
```
open/enable/开启    涩图/色图/setu    
```
在本群开启色图

```
close|disable|关闭   涩图 / 色图 / setu
```
在本群关闭色图

### 请求色图

色图两种请求方式

第一种:
```
色图 / 涩图 /setu  (tag)
```
第二种直接写正则了:   `来(\d)*(张|份)([\w\d]+)?\s*(的)?\s*(色图|涩图)\s*$`

简单来说 `来张拉克丝色图`

**<font color='red'>色图功能默认关闭，仅有机器人的管理员可以开启</font>**

***<font color='blue'>色图很容易超时</font>***
